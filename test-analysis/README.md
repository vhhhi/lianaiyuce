# 测评报告系统

一个基于React的测评报告展示系统,支持个人测评报告和匹配分析报告的展示。

## 功能特性

- 📊 个人测评报告展示
- 🔄 匹配度分析
- 📈 数据可视化
- 📱 响应式设计
- 🌐 多语言支持
- 📥 PDF导出
- 📤 报告分享

## 技术栈

- React 18
- TypeScript 4
- Ant Design 5
- React Router 6
- Echarts 5
- Jest + Testing Library
- Cypress
- GitHub Actions

## 开发指南
